**Product Opener** is the server software for **Open Food Facts** and its sibling projects (Open Beauty Facts, Open Pet Food Facts, Open Products Facts). It is released under the AGPL license and is being developed in Perl, HTML and JavaScript as [Free and Open-Source Software](https://en.wikipedia.org/wiki/Free_and_open-source_software).

<Callout type="info">
  Product Opener handles product data storage, API endpoints, user management,
  image processing, and serves the main website frontend for millions of users
  worldwide.
</Callout>

## What is Product Opener?

Product Opener is the heart of the Open Food Facts ecosystem, providing:

<Steps>
  <Step title="Web Server">
    Serves the main Open Food Facts website interface for browsing and editing
    products
  </Step>
  <Step title="Database System">
    Stores comprehensive product information, images, and user contribution data
  </Step>
  <Step title="API Backend">
    Provides REST and GraphQL APIs for third-party applications and mobile apps
  </Step>
  <Step title="Image Processing">
    Handles product image uploads, transformations, and OCR processing
  </Step>
  <Step title="User Management">
    Manages user accounts, authentication, and contribution tracking
  </Step>
  <Step title="Data Pipeline">
    Processes, validates, and enriches product information from multiple sources
  </Step>
</Steps>

## Key Features

<Accordions type="single">
  <Accordion title="Product Management" id="product-management">
    Product Opener provides comprehensive tools for managing product data:
    - **Add and Edit Products**: Users can add new products and edit existing information
    - **Upload and Manage Images**: Support for multiple product images with automatic processing
    - **Collaborative Editing**: Multiple users can contribute to the same product with full history tracking
    - **Data Quality Validation**: Automated validation and scoring of product information
    - **Batch Operations**: Tools for bulk editing and data import/export
  </Accordion>
  
  <Accordion title="Search & Discovery" id="search-discovery">
    Advanced search capabilities to help users find products:
    - **Full-text Search**: Search across all product fields including ingredients, brands, and categories
    - **Advanced Filtering**: Filter by categories, labels, ingredients, nutrition scores, and more
    - **Faceted Search Interface**: Interactive filters that show available options and counts
    - **Geographic Data Analysis**: Location-based product availability and regional preferences
    - **Personalized Results**: Customizable sorting and ranking based on user preferences
  </Accordion>
  
  <Accordion title="Data Export & Integration" id="data-export">
    Multiple ways to access and export product data:
    - **JSONL Exports**: Complete database exports in JSON Lines format for research and analysis
    - **CSV Exports**: Structured data exports for specific data sets and spreadsheet analysis
    - **API Access**: Real-time data access through REST and GraphQL APIs
    - **Integration Tools**: SDKs and tools for integrating with external systems
    - **Research Platform**: Dedicated access for academic and research institutions
  </Accordion>
</Accordions>

## Technology Stack

<Accordions type="single">
  <Accordion title="Backend Technologies" id="backend-tech">
    - **Language**: Perl with modern web frameworks and libraries
    - **Database**: MongoDB for flexible document storage and product data
    - **Cache Layer**: Redis for high-performance caching and session management
    - **Message Queue**: Background job processing for image processing and data imports
    - **File Storage**: Distributed file system for product images and documents
  </Accordion>
  
  <Accordion title="Search & Analytics" id="search-analytics">
    - **Search Engine**: Elasticsearch for full-text search and complex queries
    - **Analytics**: Real-time analytics and reporting on product data and usage
    - **Indexing**: Automated indexing pipeline for product updates and new additions
    - **Aggregations**: Fast aggregation queries for statistics and insights
  </Accordion>
  
  <Accordion title="Infrastructure & Deployment" id="infrastructure">
    - **Web Server**: Apache/Nginx for high-performance web serving
    - **Load Balancing**: Multiple server instances with traffic distribution
    - **CDN**: Global content delivery network for images and static assets
    - **Monitoring**: Comprehensive monitoring and alerting systems
    - **Backup**: Automated backup and disaster recovery procedures
  </Accordion>
</Accordions>
