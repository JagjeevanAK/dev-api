# Knowledge Panel System

<Callout type="info">
  The Knowledge Panel system provides structured, ready-to-display information
  about food products through facets, making complex product data accessible and
  understandable for users worldwide.
</Callout>

## What are Knowledge Panels?

Knowledge Panels are structured information displays that present product data in an organized, user-friendly format. They help users understand various aspects of food products through categorized information based on different facets like categories, brands, ingredients, and nutritional data.

## Key Features

<Steps>
  <Step title="Facet-Based Organization">
    Information is organized by facets such as categories, labels, ingredients,
    nutritional aspects, and environmental impact
  </Step>
  <Step title="Multi-Language Support">
    Knowledge panels support multiple languages with localized content for
    international accessibility
  </Step>
  <Step title="Country-Specific Data">
    Panels display region-specific information based on country parameters,
    regulations, and local preferences
  </Step>
  <Step title="API Integration">
    Easy integration with external applications through a comprehensive RESTful
    API
  </Step>
  <Step title="Real-time Updates">
    Dynamic content that updates based on the latest product information and
    scientific data
  </Step>
</Steps>

## System Architecture

### Core Components

- **FastAPI Backend**: High-performance API service for serving knowledge panels
- **Facet Engine**: Processes and structures product data into meaningful categories
- **Localization System**: Handles multi-language content and country-specific information
- **Caching Layer**: Optimizes performance for frequently accessed panels

### Integration Points

- **Product Opener**: Connects with the main product database
- **Robotoff**: Integrates AI-generated insights and predictions
- **External APIs**: Supports third-party applications and mobile apps

## API Usage

The Knowledge Panel API allows you to retrieve structured information about products based on various facets. Here's a quick example:

```json title="Example API Request"
GET https://facets-kp.openfoodfacts.org/knowledge_panel?facet_tag=category&value_tag=yogurts&lang_code=en&country=us
```
